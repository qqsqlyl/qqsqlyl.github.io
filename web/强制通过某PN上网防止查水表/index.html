强制通过VPN上网，VPN断开链接即断网

web
发布时间 : 2020-11-05 22:03
字数:275
  1. 0x1：默认阻止所有出口流量：
  2. 0x2：允许通向VPN服务器的流量
  3. 0x3：允许流量通过vpn链接出去

0x1：默认阻止所有出口流量：

###1.首先开启防火墙：

###2.打开 高级设置 ===> Windows Defender 防火墙属性 ===>1.域配置文件(1.防火墙状态启用2.出站连阻止) 2.专用配置文件(1.防火墙状态启用2.出站连阻止) 3.公用配置文件(1.防火墙状态启用2.出站连阻止) 一共三个都要设置

0x2：允许通向VPN服务器的流量

###1.在出站规则===>右键新建规则===>规则类型 自定义(C)

###程序 协议和端口都默认 ====> 1.规则默认(任何IP地址)===>2.规则选下列IP地址 添加VPN的IP

###下一步到操作选择(允许链接A) 配置文件（选1.域2.专用3.公用）都默就行 填好名称完成。

0x3：允许流量通过vpn链接出去

###1.步骤与上一步类似 修改地方 在作用域都选择任何IP 然后点击===>自定义===>下列接口类型===>远程访问

###XPV 断开会禁止网络出口
